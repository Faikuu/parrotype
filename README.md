## Features
- React/Nest stack
- Translation support (Currently only english and polish is available out of the box)
- Individual models per language
- Saving user's performance to database and displaying it on Account Page.

## Usage/Examples
Installing
```sh
npm run initialize
```
Development
```sh
npm run start:dev
```
Production
```sh
npm run start
```
